Le projet

Pour mettre en pratique mes nouvelles compétences en React j'ai crée une application "to-do-list" ou liste des tâches qui permet d'ajouter et de supprimer des tâches. C'est une application plutôt classique - chaque fois qu'on apprend un nouveau framework on crée souvent ce type d'appli parce qu'elle permet de comprendre le fonctionnement basique d'un CRUD.

La mission

L'application est composé par deux composants, addForm et Todo. Le premier s'occupe de récuperer la valeur renseigné par l'utilisateur dans le champ pour ensuite le passer à App.js. Dans ce fichier ce trouver le state et la fonction AddToDo qui a le but de recupérer les informations passées par le composant et modifier le state. Pour ce qui concerne le composant Todo il fait une simple boucle (avec la méthode .map) sur les éléments contenus dans le state pour les visualiser.
