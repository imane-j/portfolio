ISS

Type de projet : Carte d'information

Objectif : Représenter des données dynamiques sur une
carte en utilisant une API

Outils utilisés : Javascript (ES6 avec les promesses),
Leaflet.js, Mapbox pour les tiles, API "Where the ISS at"

Lien vers le site

Le projet

Pendant l'été j'ai décidé d'apprendre toutes les nouvelles particularités de la version ES6 de Javascript. Dans le cadre de cet apprentissage j'ai réalisé une carte capable de montrer la position exacte de la station spatiale internationale grâce à l'API "Where the ISS at?".

Stack technique

La connexion et récupération des données à été réalisé en utilisant les promesses, une nouvelle fonctionnalité de ES6 qui facilite la communication avec les API. Pour ce qui concerne la carte elle a été réalisé avec la librairie Leaflet.js. Pour la personnalisation graphique j'ai utilisé l'API de Mapbox.
